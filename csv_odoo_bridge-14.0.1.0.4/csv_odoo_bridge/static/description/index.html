Coming to the business management; what could be better than a software which has a lot for your business.Odoo is a software which offers you a complete business management solution. Odoo is widely known for its unmatched inventory management, accounting, erp and much more; summing up, it lets you manage all your business data.

You may need to share the data in your team further; in that case, a format which is supportive in all environments is a must. If you need to share your Odoo data to others in your team then why not use CSV format.

A solution for Odooers who are thinking of managing the data in a file format which is quick and easy yet safe; Odoo Multichannel CSV Connector allows you to easily import data from CSV to Odoo.

You can also connect to multiple e-commerce and marketplaces to your single Odoo instance with the help of Odoo Multi-channel Sale.

Importing Orders in Odoo

You can import Orders along with the data associated with it from CSV to Odoo.

The import function is simplified with the help of module Odoo Multichannel CSV Connector.

Orders statuses like cancel, shipment and invoice are automatically imported at the same time.

Catalog data is also imported at the time of importing orders.

Moreover, data like customers and their addresses, taxes, delivery methods, payment methods, currency is also imported along with it.

What do you get extra from the default flow of Importing data in Odoo?

In the default process of Odoo, you can only import the separated data at a time. For instance, while importing Orders you can only import the Order data but not the associated data.

Odoo Multichannel CSV Connector offers you ease of importing all the associated data together.

Key Features

Below is the detailed list of features from Odoo MultiChannel CSV Connector

Importing Orders in Odoo

  • Orders are manually imported in bulk
  • You can import orders via cron scheduler
  • Order details (like customers and their addresses, taxes, delivery methods, payment methods, currency) are imported along with the orders
  • Order statuses (like cancel, shipment and invoice) are also imported along with the orders

Enhanced Dynamic Dashboard

  • Introduced a dynamic dashboard to track the data of CSV to your Odoo.
  • Import/Export the data of any channel directly from the dashboard.
  • Well-designed Donut Chart and Line Graph to analyze the data.
  • The dashboard also offers Clickable tabs and data cards.

Manually Map CSV Headers

  • You can manually map header along with the Odoo feed fields

Managing Synchronization History

  • All the synchronization history is managed in Odoo

Manually Map Orders Statuses

  • You can manually map order statuses from Odoo end
  • Fill the corresponding Order status of CSV and Odoo in the Order Status table

Donut Chart to Analyze the data

  • Drop down to Select the Data Type (Order and Product)
  • Track the data of connected CSV instance from the chart

Line Graph to Analyze the data

  • Drop down to Select the Data Type (Order, Product, Category and Customer)
  • Track the data the Line Graph

Clickable Data Cards for the Details

  • View the imported Product, Category, Order and Customers in the Tabs
  • The tabs are clickable and direct you to the respective mapping, feeds as well as sync history.
